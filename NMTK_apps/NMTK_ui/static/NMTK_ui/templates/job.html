Jobs

Here you can create new jobs by pairing a data file with a tool and then providing some configuration parameters.

When a job is running, you can review it's current status on this tab, as reported by the tool server. Once the job is complete, you can view it's results or download them for further analysis. Job results are also stored in your Files library and you can retrieve them from there as well, or use them as input to other jobs.
