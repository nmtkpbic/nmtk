Jobs

Here you can create new analysis jobs by pairing a data file with a tool and then providing some configuration parameters.

When an analysis job is running, you can see its current status hare, or pop up a detailed list of all the status messages from "Actions" popup for your job. Once the analysis is complete, you can view it's results or download them for further analysis. Analysis results are also stored in your Files library and you can retrieve them from there as well, or use them as input to other analysis jobs.
