Nonmotorized Travel Analysis Toolkit Introduction

The Nonmotorized Travel Analysis Toolkit is a growing suite of software applications that support pedestrian and bicycle transportation planning.

The Toolkit offers a number of analysis tools that you can apply to your own data, including models for demand estimation on roadway segments (or "links") and at intersections.

These are the steps for using the Toolkit:

  • Begin by exploring the analysis tools (below):
    • Read the tool documentation to learn about what each tool does and how it was developed.
    • Familiarize yourself with the data required for each tool by loading and examining the tool's sample data and sample job.
  • Prepare and upload the necessary data for your selected tool.
  • Start a new analysis job, filling in the necessary information.
  • Submit the analysis (a "job") to run it (your data will be uploaded to the tool, and you will be notified when the results are ready.)
  • View the results, which can be accessed in one of the following ways:
    • In your browser (for spatial data, you can see it on a map!)
    • As a Microsoft Excel Spreadsheet (XLS) file.
    • As a Comma-Separated Values (CSV) file.
    • As a GeoJSON file (the "native" format for the Toolkit)

Below is a list of currently available tools. It is expected that new modeling and anslysis tools will be added in the future. Select a tool to read about it and try it out. When you are ready to apply the analysis tool to your own data, click on the Jobs tab to create and run your own customized analysis.
