Files

Here you can manage your data files, and upload your own data for use in a tool. Once stored, a data file can be used as the input for one or more analysis jobs. Use the Tool Explorer to find out what data each tool needs. You can set up analysis jobs from the Jobs tab.

You can upload many data formats, including CSV, DBF, and Excel files. If one of those tabular formats includes longitude/latitude values, it can also be mapped. Common geographic vector formats are also supported for upload. To upload a shapefile, put all of its components in a single .zip archive. The .zip file will be unpacked automatically and the shapefile loaded.

If you want to use the output from one analysis job as the input to another, the results from each analysis are stored here as well and can be used just like any uploaded file.

Once you've uploaded a file, you can use its "Action" dropdown to add a description, view the file or its metadata, or retrieve the file.

Add New Files
